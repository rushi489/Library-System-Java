# 📚 Online Library System 

🔹 Overview
This is a Java-based Online Library System developed using Servlets and JDBC. It allows users to register, login, and manage their accounts.

🔹 Features
* User Registration
* User Login Authentication
* Password Reset
* Database Integration using JDBC

🔹 Tech Stack
* Java (Core + OOP)
* Servlets (Jakarta EE)
* JDBC
* MySQL
* HTML/CSS
* Apache Tomcat

🔹 Project Structure
* `src/` → Java source files (Servlets)
* `WebContent/` → Frontend files (JSP/HTML)
* `images/` → UI assets

🔹 How to Run
1. Import project into Eclipse/IntelliJ
2. Configure Apache Tomcat server
3. Set up MySQL database
4. Update DB credentials in code
5. Run the project on server

🔹 Future Improvements
* Password hashing (security)
* MVC architecture (DAO layer)
* Book issue/return feature
* Admin dashboard
